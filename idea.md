# Brand Monitor AI
is a next generation gen-z platform to find what people asking about brands to LLMs (Grok, Gemini, ChatGPT, DeepSeek, Kimi, etc). The entire user experience must match with gen-z vibes, so, gen-z can promote their brands or get information about the favorite brands to build up next viral or trending things according to upcoming insights from popular LLMs (Grok, Geimini, ChatGPT, DeepSeek, Kimi, etc)

1. Next.js web framework (AI SEO First)
2. Typescript
3. Tailwind CSS, DaisyUI, RadixUI, Lucide Icons
4. OpenAI SDK or AI SDK or any related AI SDK 
5. Cybersecurity first
6. Gen-Z UX UI Design Experience, 
7. Beautiful Animations 
8. Bun.js Package Manager

## Features
1. User can search any brand by name
2. User Login or Register (Authentication complete flow)
3. User can add brands to their watchlist
4. User can get notifications when people ask about their brand
5. User can get insights about their brand
6. User can get analytics about their brand
7. User can get recommendations about their brand
8. User can share public link with their friends (PDF or Word or Excel (depth insights))
